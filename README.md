# Coyotw-Python-and-JSON
Sistema de python com salvamento em .JSON.

O código até o momento atualizado está sendo capaz de:

- Adcionar usuarios
- Remover usuarios
- Listar usuarios
- Salvamento automatico da lista em .json

Foram estudados os seguintes conceitos dentro desse projeto:

- Def
- Listas
- Dicionarios
- if/else/elif
- JavaScript Object Notation
- import
- Try/Except
- Loops (básico)
- Enumerate
- Len

Sempre atualizando com o conhecimento que vou adquirindo com o passar dos dias.
